# Container images / Using Docker Hub images #

* go to [hub.docker.com](hub.docker.com)
* search for **nginx**
* result must be a single name and has **official** type

Download latest:

    docker pull nginx

Download specific version:

    docker pull nginx:1.10.3

See images list:

    docker image ls
